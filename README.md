این پروژه صفحه فرود (Landing Page) رسمی AmicoChat است، یک دوست مجازی AI برای مکالمه و همدمی.
ویژگی‌ها

    طراحی مدرن با TailwindCSS و رنگ‌های گرم و دلنشین.
    نمایش لوگو و برندینگ AmicoChat.
    دکمه "دوست مجازی" که به آدرس اصلی چت هدایت می‌شود.
    بخش کوتاه معرفی قابلیت‌های دوست مجازی.
    فوتر شامل شبکه‌های اجتماعی و حقوق نشر.
